# Exploring Uninformed Search Methods

A virtual AI lab notebook implementing and visualizing classic uninformed (blind) search algorithms in Python, with clear code and interactive visualizations. Perfect for students, researchers, and AI enthusiasts.

---

## üö© About

- **Focus**: Uninformed search algorithms (no heuristics).
- **Goal**: Clear, educational Python implementations with step-by-step graph visualizations.
- **Tools**: `python`, `matplotlib`, `networkx`.

## ‚ñ∂Ô∏è Usage

- Each algorithm = its own script (e.g., `bfs.py`, `dfs.py`, etc.)
- Run with:

  ```bash
  python bfs.py
  ```

- Scripts:
  - Build example graphs
  - Run algorithm from start to goal node
  - Print results
  - Show matplotlib step-by-step visualization
